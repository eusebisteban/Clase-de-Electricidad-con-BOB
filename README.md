# Proyecto de Análisis y Creación de Presentaciones - Electromagnetismo

Este repositorio contiene scripts de Python para analizar y crear presentaciones de PowerPoint sobre temas de Electromagnetismo, específicamente sobre Transformadores.

## 📁 Contenido del Proyecto

### Scripts de Python
- **`analizar_presentaciones.py`** - Script para analizar la estructura de presentaciones existentes
- **`analizar_presentaciones_v2.py`** - Versión mejorada del analizador de presentaciones
- **`extraer_pdf_transformadores.py`** - Extrae contenido de PDFs sobre transformadores
- **`crear_presentacion_transformadores.py`** - Crea presentaciones sobre transformadores
- **`crear_presentacion_con_tema.py`** - Crea presentaciones con temas personalizados

### Documentación
- **`ANALISIS_ESTRUCTURA_CLASES.md`** - Análisis de la estructura de las clases
- **`GUIA_RAPIDA_CREACION_PRESENTACIONES.md`** - Guía rápida para crear presentaciones
- **`S16_Transformadores_ESTRUCTURA.md`** - Estructura del tema de transformadores
- **`S16_Transformadores_CONTENIDO.txt`** - Contenido textual sobre transformadores
- **`S16_Transformadores_FORMULAS.txt`** - Fórmulas relacionadas con transformadores

### Archivos de Datos
- **`analisis_presentaciones.json`** - Resultados del análisis en formato JSON
- **`analisis_presentaciones.txt`** - Resultados del análisis en formato texto
- **`transformadores_extraido.txt`** - Contenido extraído de PDFs

## 🚀 Uso

### Requisitos
```bash
pip install python-pptx
```

### Analizar Presentaciones
```bash
python analizar_presentaciones_v2.py
```

### Crear Presentación sobre Transformadores
```bash
python crear_presentacion_transformadores.py
```

## 📚 Tema: Transformadores

El proyecto se enfoca en la creación de material educativo sobre transformadores eléctricos, incluyendo:
- Principios de funcionamiento
- Fórmulas y ecuaciones
- Aplicaciones prácticas
- Análisis de circuitos
