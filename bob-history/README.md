## 📄 Archivos en cada conversación

Cada carpeta contiene:

- **`api_conversation_history.json`** - Historial completo de la conversación con la API (incluye todos los mensajes, herramientas usadas, y resultados)
- **`task_metadata.json`** - Metadatos de la tarea (archivos en contexto, fechas de modificación)
- **`ui_messages.json`** - Mensajes mostrados en la interfaz de usuario

## 🔍 Contenido de los archivos JSON

### api_conversation_history.json
Contiene el historial completo incluyendo:
- Mensajes del usuario
- Respuestas de Bob
- Herramientas utilizadas (read_file, write_to_file, execute_command, etc.)
- Resultados de comandos ejecutados
- Archivos creados o modificados

### task_metadata.json
Contiene información sobre:
- Archivos que estuvieron en contexto durante la conversación
- Estado de cada archivo (active, stale)
- Fechas de lectura y edición

### ui_messages.json
Contiene los mensajes formateados para la interfaz de usuario
